# О компании

**Oviva** — digital health компания, лидер в behavior-change терапии в Европе. Работают с диабетом, ожирением и другими хроническими заболеваниями.

**Tech Stack:**
- Java Microservices, REST APIs, Messaging, OAuth2
- Quarkus
- Google Cloud Platform, Kubernetes, Docker

**GitHub:** [oviva-ag](https://github.com/oviva-ag) — фокус на German healthcare infrastructure

# Процесс интервью

По отзыву Staff Backend Engineer (Berlin, Oct 2024):

1. Technical Recruiter Call — 1 час
2. **Technical Interview + High Level System Design** — 1 час
3. Meet the team + Behaviour — 1 час
4. Meet the VP — 20 минут

# Вопросы с интервью

**Технические:** Java/OOP, System Design, Multithreading, Bug fixing, Time complexity, SQL basics

**Behavioral:** "Tell me about a time when...", конфликты, челленджи, почему Oviva
